## 📦 بخش انبارداری (Inventory)

### آمار نمایش داده شده:
```typescript
✅ تعداد کالا: products.length
✅ موجودی کل: sum(product.stock)
✅ هشدار موجودی: count(stock < minStockAlert)
✅ ارزش خرید: sum(stock × buyPrice)
✅ ارزش فروش: sum(stock × sellPrice)
✅ سود بالقوه: ارزش فروش - ارزش خرید
```

### فیلترها:
```typescript

✅ مرتب‌سازی: پیش‌فرض / آخرین تغییر قیمت
```

### نمایش موجودی:
```typescript
stock === 0  => قرمز (ناموجود)
stock < minStockAlert => زرد (کمبود)
stock >= minStockAlert => سبز (موجود)
```
