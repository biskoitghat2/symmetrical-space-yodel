## 📊 بررسی کاردکس مشتری (Customer Cardex)

### 🔍 منطق محاسبات بدهکار/بستانکار

#### قوانین اصلی:
```
بدهکار (isDebtor = true):  مشتری به ما بدهکار است → مانده مثبت
بستانکار (isDebtor = false): ما به مشتری بدهکار هستیم → مانده منفی

فرمول محاسبه:
effect = isDebtor ? +amount : -amount
newBalance = currentBalance + effect
```
