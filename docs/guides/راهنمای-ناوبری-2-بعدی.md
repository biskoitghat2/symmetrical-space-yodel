## ⌨️ کلیدهای میانبر

### در فیلد جستجو:
- `⬆️ Arrow Up`: حرکت در لیست dropdown
- `⬇️ Arrow Down`: حرکت در لیست dropdown
- `Enter`: افزودن کالا → فوکوس روی تعداد
- `Esc`: بستن dropdown

### در جدول (تعداد/قیمت/تخفیف):
- `⬆️ Arrow Up`: رفتن به ردیف بالا (همان ستون)
- `⬇️ Arrow Down`: رفتن به ردیف پایین (همان ستون)
- `➡️ Arrow Right`: رفتن به ستون قبلی (در RTL)
- `⬅️ Arrow Left`: رفتن به ستون بعدی (در RTL)
- `Enter`: رفتن به ستون بعدی
- `Backspace` (در فیلد خالی): برگشت به ستون قبلی

---

## 🎯 مسیرهای ناوبری

### مسیر افقی (چپ/راست):
```
تعداد → (Enter/Left) → قیمت → (Enter/Left) → تخفیف
تعداد ← (Right) ← قیمت ← (Right) ← تخفیف
```

### مسیر عمودی (بالا/پایین):
```
ردیف 1: تعداد
    ↕️ (Up/Down)
ردیف 2: تعداد
    ↕️ (Up/Down)
ردیف 3: تعداد
```

### مسیر ترکیبی:
```
جستجو → Enter → تعداد[1,1]
                    ↓ Down
                 تعداد[2,1]
                    → Left
                 قیمت[2,2]
                    ↓ Down
                 قیمت[3,2]
                    → Left
                 تخفیف[3,3]
                    → Enter
                 جستجو
```

## 💡 ویژگی‌های هوشمند

### 1. Auto-Select
```typescript
onFocus={e => e.target.select()}
```
- با فوکوس شدن، تمام متن انتخاب می‌شود
- تایپ مستقیم، مقدار قبلی را پاک می‌کند

### 2. Edge Handling
- بالای اولین ردیف → جستجو
- پایین آخرین ردیف → همان ردیف
- راست اولین ستون → جستجو
- چپ آخرین ستون → ردیف بعدی یا جستجو

### 3. RTL Support
- `Arrow Right` = ستون قبلی (تعداد ← قیمت)
- `Arrow Left` = ستون بعدی (تعداد → قیمت)

## 🔧 جزئیات فنی

### Grid Structure:
```typescript
Columns:
  0 = Quantity (تعداد)
  1 = Price (قیمت فی)
  2 = Discount (تخفیف)

Rows:
  0 to N = formState.items
```

### Navigation Function:
```typescript
navigateGrid(itemId, currentCol, direction)
  - itemId: شناسه آیتم فعلی
  - currentCol: 0, 1, or 2
  - direction: 'up' | 'down' | 'left' | 'right'
```

### Refs Structure:
```typescript
quantityRefs.current[itemId] → HTMLInputElement
priceRefs.current[itemId] → HTMLInputElement
discountRefs.current[itemId] → HTMLInputElement
```

## 📊 مثال عملی

### سناریو: ثبت 3 کالا
```
1. تایپ "لپ" → Enter
   → فوکوس: تعداد[1,0]

2. تایپ "2" → Enter
   → فوکوس: قیمت[1,1]

3. Enter (قیمت پیش‌فرض تایید)
   → فوکوس: تخفیف[1,2]

4. تایپ "100000" → Enter
   → فوکوس: تعداد[2,0] (ردیف بعدی)

5. تایپ "موب" در جستجو → Enter
   → فوکوس: تعداد[2,0]

6. Arrow Up
   → فوکوس: تعداد[1,0] (ردیف قبلی)

7. Arrow Left
   → فوکوس: قیمت[1,1] (ستون بعدی)

8. Arrow Down
   → فوکوس: قیمت[2,1] (ردیف بعدی، همان ستون)
```

**زمان کل: ~20 ثانیه! ⚡**
