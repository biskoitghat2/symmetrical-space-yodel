## 🔍 تشخیص مشکل

### مشکل شما: "هیچی سیو نمیشه و نوتیف نمیاد"

این مشکل می‌تواند 3 دلیل داشته باشد:

### دلیل 1: دیتابیس اصلاً initialize نشده ❌
**چک کنید**: Console را باز کنید (F12) و ببینید این پیام‌ها هست یا نه:
```
🔄 Loading database...
✅ Database loaded
✅ Tables created
✅ Database initialized successfully
```

**اگر این پیام‌ها نیست**:
- برنامه در مرورگر اجرا شده (نه Tauri)
- باید با `npm run tauri dev` اجرا کنید

### دلیل 2: دیتابیس initialize شده ولی save نمیکنه ⚠️
**چک کنید**: وقتی کالا اضافه می‌کنید، در Console خطایی می‌بینید؟

**اگر خطا هست**:
- خطای SQL یا Database
- لطفاً متن کامل خطا را به من بدهید

**اگر خطا نیست**:
- داده در دیتابیس save میشه ولی بعد از بستن برنامه پاک میشه
- این یعنی مشکل در مسیر فایل دیتابیس است

### دلیل 3: نوتیفیکیشن شرایطش برقرار نیست ℹ️
نوتیفیکیشن فقط در این شرایط نمایش داده میشه:

**نوتیفیکیشن موجودی**:
- موجودی کالا = 0 → "اتمام موجودی" (قرمز)
- موجودی کالا ≤ حد هشدار → "هشدار موجودی کم" (زرد)

**نوتیفیکیشن چک**:
- چک سررسید امروز → "سررسید چک امروز" (زرد)
- چک سررسید گذشته → "چک معوقه" (قرمز)

**نوتیفیکیشن مشتری**:
- بدهی مشتری > 500,000,000 ریال → "سقف بدهی مشتری" (آبی)

## 🧪 تست‌های لازم

### تست 1: آیا دیتابیس کار میکنه؟
```
1. برنامه را با npm run tauri dev اجرا کنید
2. F12 را بزنید و Console را باز کنید
3. به بخش "مدیریت انبار" بروید
4. یک کالای جدید با نام "تست-دیتابیس" اضافه کنید
5. برنامه را ببندید (دکمه X)
6. دوباره برنامه را باز کنید
7. آیا کالای "تست-دیتابیس" هنوز وجود دارد؟
```

**نتیجه مورد انتظار**:
- ✅ اگر کالا هنوز هست → دیتابیس کار میکنه
- ❌ اگر کالا نیست → مشکل در save کردن است

### تست 2: آیا نوتیفیکیشن کار میکنه؟
```
1. یک کالا با موجودی 0 اضافه کنید
2. به آیکون زنگ (🔔) در گوشه بالا نگاه کنید
3. روی آیکون کلیک کنید
```

**نتیجه مورد انتظار**:
- ✅ باید نوتیفیکیشن "اتمام موجودی" نمایش داده شود
- ❌ اگر نوتیفیکیشن نیست → مشکل در سیستم نوتیفیکیشن است

### تست 3: آیا برنامه فوری میبنده؟
```
1. کالایی اضافه کنید
2. بلافاصله دکمه X را بزنید
```

**نتیجه مورد انتظار**:
- ✅ برنامه باید فوراً بسته شود (کمتر از 1 ثانیه)
- ❌ اگر 5-10 ثانیه فریز میکنه → مشکل قدیمی هنوز هست

## 🔧 راه‌حل‌های احتمالی

### اگر دیتابیس initialize نمیشه:
```bash
# مطمئن شوید Tauri plugin نصب است
cd src-tauri
cargo build
```

### اگر دیتابیس initialize میشه ولی save نمیکنه:
مشکل احتمالاً در permissions یا مسیر فایل است. باید Console logs را ببینم.

### اگر نوتیفیکیشن نمیاد:
شرایط نوتیفیکیشن را چک کنید (موجودی صفر، چک سررسید، بدهی بالا)

4. اگر خطایی هست، متن کامل آن را بفرستید
