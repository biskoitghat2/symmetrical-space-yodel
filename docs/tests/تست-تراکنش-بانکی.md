## ✅ بررسی منطق processBankTransaction

### کد بررسی شده:
```typescript
processBankTransaction: (transaction) => {
    // 1. Transfer: کسر از مبدا، اضافه به مقصد
    if (transaction.type === 'transfer') {
        source.balance -= transaction.amount;
        dest.balance += transaction.amount;
    }
    
    // 2. Income: اضافه به حساب
    else if (transaction.type === 'income') {
        account.balance += transaction.amount;
    }
    
    // 3. Expense: کسر از حساب
    else if (transaction.type === 'expense') {
        account.balance -= transaction.amount;
    }
    
    // 4. بروزرسانی مانده مشتری (اگر مشتری انتخاب شده باشد)
    if (transaction.customerId && transaction.type !== 'transfer') {
        const isDebtor = transaction.type === 'expense';
        const effect = isDebtor ? +amount : -amount;
        customer.balance += effect;
    }
}
```
