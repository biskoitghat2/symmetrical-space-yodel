# 📁 راهنمای ذخیره‌سازی دیتا

## 📍 مسیر ذخیره دیتا

دیتای برنامه HesabFlow در مسیر زیر ذخیره می‌شود:

### ویندوز:
```
C:\Users\[نام کاربری]\AppData\Roaming\com.hesabflow.app\
```

**مثال:**
```
C:\Users\Ali\AppData\Roaming\com.hesabflow.app\hesabflow-data.json
```

---

## 📂 ساختار فایل‌ها

```
C:\Users\[نام کاربری]\AppData\Roaming\com.hesabflow.app\
│
├── hesabflow-data.json          ← فایل اصلی دیتا
│
└── backups\                     ← پوشه نسخه‌های پشتیبان
    ├── backup_2026-02-19T10-30-00.json
    ├── backup_2026-02-19T11-00-00.json
    └── backup_2026-02-19T11-30-00.json
```

## 💾 مدیریت دیتا

### 1️⃣ مشاهده مسیر دیتا

از داخل برنامه:
```
تنظیمات → بخش "مدیریت دیتا" → مسیر فایل دیتا
```

### 2️⃣ Export دیتا (خروجی گرفتن)

```
تنظیمات → دکمه "خروجی دیتا" → انتخاب مسیر → ذخیره
```

**استفاده:**
- 📤 انتقال به کامپیوتر دیگر
- 💾 نسخه پشتیبان دستی
- 📧 ارسال به پشتیبانی

### 3️⃣ Import دیتا (وارد کردن)

```
تنظیمات → دکمه "ورود دیتا" → انتخاب فایل → تایید
```

**هشدار:** ⚠️ دیتای فعلی جایگزین می‌شود!

### 4️⃣ Backup دستی

```
تنظیمات → دکمه "نسخه پشتیبان" → فایل در پوشه backups ذخیره می‌شود
```

## 🔍 دسترسی به پوشه دیتا

### روش 1: از طریق Run
```
1. کلید Windows + R
2. تایپ کنید: %APPDATA%
3. Enter
4. پوشه com.hesabflow.app را باز کنید
```

### روش 2: از طریق File Explorer
```
1. File Explorer را باز کنید
2. در آدرس بار تایپ کنید:
   %APPDATA%\com.hesabflow.app
3. Enter
```

### روش 3: از طریق برنامه
```
تنظیمات → "باز کردن پوشه دیتا" (اگر دکمه وجود داشته باشد)
```

---

## 🛡️ امنیت دیتا

### محافظت از دیتا:

1. **نسخه پشتیبان منظم:**
   ```
   - هر هفته یک نسخه Export بگیرید
   - در مکان امن ذخیره کنید (فلش، کلود، ...)
   ```

2. **قبل از بروزرسانی:**
   ```
   - حتماً Export بگیرید
   - نسخه پشتیبان را نگه دارید
   ```

3. **قبل از حذف برنامه:**
   ```
   - Export دیتا
   - یا کپی دستی از پوشه com.hesabflow.app
   ```

---

## 🔄 انتقال به کامپیوتر دیگر

### مرحله 1: Export از کامپیوتر قدیم
```
1. برنامه را باز کنید
2. تنظیمات → خروجی دیتا
3. فایل را در Desktop ذخیره کنید
4. فایل را به کامپیوتر جدید منتقل کنید
```

### مرحله 2: Import در کامپیوتر جدید
```
1. برنامه را نصب کنید
2. برنامه را باز کنید
3. تنظیمات → ورود دیتا
4. فایل Export شده را انتخاب کنید
```

## 🗑️ حذف دیتا

### حذف کامل دیتا:

**روش 1: از داخل برنامه**
```
تنظیمات → "پاک کردن تمام دیتا" (اگر وجود داشته باشد)
```

**روش 2: حذف دستی**
```
1. برنامه را ببندید
2. به پوشه بروید:
   %APPDATA%\com.hesabflow.app
3. فایل hesabflow-data.json را حذف کنید
4. برنامه را دوباره باز کنید (دیتا خالی)
```

**هشدار:** ⚠️ این عمل غیرقابل بازگشت است!

## 🔧 عیب‌یابی

### مشکل: دیتا ذخیره نمی‌شود

**راه حل:**
```
1. بررسی مجوزهای پوشه:
   - پوشه AppData باید قابل نوشتن باشد
   
2. بررسی فضای دیسک:
   - حداقل 100 MB فضای خالی

3. بررسی آنتی‌ویروس:
   - ممکن است دسترسی را مسدود کرده باشد
```

### مشکل: فایل دیتا پیدا نمی‌شود

**راه حل:**
```
1. مطمئن شوید برنامه حداقل یکبار اجرا شده
2. یک تراکنش ثبت کنید و برنامه را ببندید
3. دوباره به پوشه نگاه کنید
```

### مشکل: دیتا بعد از بروزرسانی از بین رفت

**راه حل:**
```
1. به پوشه backups بروید:
   %APPDATA%\com.hesabflow.app\backups
   
2. آخرین فایل backup را پیدا کنید

3. از Import استفاده کنید یا:
   - فایل backup را rename کنید به hesabflow-data.json
   - در پوشه اصلی قرار دهید
```
