## 🧪 تست‌های لازم

### تست سریع 1: آیا دیتابیس کار میکنه؟
```
1. npm run tauri dev
2. F12 → Console
3. انبار → کالای جدید با نام "تست"
4. ببندید و دوباره باز کنید
5. آیا کالای "تست" هنوز هست؟
```

### تست سریع 2: آیا نوتیفیکیشن کار میکنه؟
```
1. کالای جدید با موجودی 0 اضافه کنید
2. آیکون زنگ (🔔) را کلیک کنید
3. آیا نوتیفیکیشن "اتمام موجودی" می‌بینید؟
```
