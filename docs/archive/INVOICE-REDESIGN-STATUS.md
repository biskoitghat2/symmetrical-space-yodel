## مشکل برطرف شد ✅

خطای `ReferenceError: searchInputRef is not defined` برطرف شد.

### تغییرات اعمال شده:

1. ✅ State های قدیمی حذف شدند
2. ✅ State های جدید اضافه شدند (activeCell, rowSearchQueries, rowDropdownStates)
3. ✅ توابع navigation جدید اضافه شدند
4. ✅ فیلد جستجوی قدیمی حذف شد
5. ✅ جدول قدیمی موقتاً غیرفعال شد

### وضعیت فعلی:

برنامه اکنون بدون خطا اجرا می‌شود، اما جدول فاکتور موقتاً یک پیام "در حال توسعه" نمایش می‌دهد.

### مراحل بعدی:

برای تکمیل بازطراحی، باید جدول جدید با ویژگی‌های زیر پیاده‌سازی شود:

1. **Inline Search در هر ردیف**
   - هر سلول "نام کالا" یک فیلد جستجوی مستقل دارد
   - Dropdown زیر همان سلول باز می‌شود

2. **ستون‌های جدید**
   - واحد (عدد، کیلو، متر، ...)
   - قیمت خرید (با پس‌زمینه زرد)
   - آخرین تغییر قیمت
   - سود (محاسبه لحظه‌ای)

3. **ناوبری 4 جهته**
   - Arrow Up/Down: بین ردیف‌ها
   - Arrow Left/Right: بین ستون‌ها
   - Enter: سلول بعدی
   - Backspace: سلول قبلی

4. **Active Row Highlighting**
   - ردیف فعال با پس‌زمینه آبی مشخص می‌شود

5. **فونت‌های بزرگتر**
   - Row Height: 60px
   - Font Size: 16-18px

### راهنمای کامل:
