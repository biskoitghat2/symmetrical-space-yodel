# ✅ بهبودهای Product Search Modal

## 🎯 تغییرات اعمال شده

### 1. ✅ نمایش تاریخ آخرین تغییر قیمت‌ها

**قبل:**
- فقط قیمت فروش و خرید نمایش داده می‌شد

**بعد:**
- ✅ ستون "قیمت فروش" + ستون "تاریخ" (آخرین تغییر قیمت فروش)
- ✅ ستون "قیمت خرید" + ستون "تاریخ" (آخرین تغییر قیمت خرید)

**فیلدهای استفاده شده:**
```typescript
product.lastSellDate  // تاریخ آخرین تغییر قیمت فروش
product.lastBuyDate   // تاریخ آخرین تغییر قیمت خرید
```

### 2. ✅ دکمه ویرایش کالا

**قابلیت جدید:**
- ✅ ستون "عملیات" اضافه شد
- ✅ دکمه ویرایش (Edit icon) در هر ردیف
- ✅ کلیک روی دکمه → باز شدن فرم ویرایش کالا
- ✅ Modal باز می‌ماند (کاربر می‌تواند بعد از ویرایش، همان کالا را انتخاب کند)

**نحوه کار:**
```typescript
onEdit={(product) => {
  openWindow('ویرایش کالا', 'PRODUCT_FORM', { product });
}}
```

### فایل: `components/ui/ProductSearchModal.tsx`

#### Props جدید:
```typescript
interface ProductSearchModalProps {
  // ... سایر props
  onEdit?: (product: Product) => void;  // ⭐ اضافه شد
}
```

#### Handler جدید:
```typescript
const handleEdit = (e: React.MouseEvent, product: Product) => {
  e.stopPropagation();  // جلوگیری از انتخاب کالا
  if (onEdit) {
    onEdit(product);
  }
};
```

## 📱 تجربه کاربری

### سناریو 1: مشاهده تاریخ تغییرات
1. کاربر Modal را باز می‌کند
2. در جدول، تاریخ آخرین تغییر قیمت‌ها را می‌بیند
3. می‌تواند تصمیم بگیرد که آیا قیمت‌ها به‌روز هستند یا نه

### سناریو 2: ویرایش سریع کالا
1. کاربر در حال انتخاب کالا است
2. متوجه می‌شود قیمت یا موجودی نیاز به ویرایش دارد
3. روی دکمه ویرایش کلیک می‌کند
4. فرم ویرایش کالا باز می‌شود
5. تغییرات را اعمال می‌کند
6. به Modal برمی‌گردد و کالا را انتخاب می‌کند
