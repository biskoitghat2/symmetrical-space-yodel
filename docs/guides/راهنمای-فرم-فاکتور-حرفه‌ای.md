# 🚀 راهنمای فرم فاکتور حرفه‌ای

## ✅ قابلیت‌های جدید پیاده‌سازی شده

### 1. جستجوی هوشمند با Dropdown

#### ویژگی‌ها:
- ✅ Dropdown با انیمیشن smooth و سریع
- ✅ Auto-scroll هوشمند با حرکت کیبورد
- ✅ Highlight روی آیتم انتخاب شده
- ✅ نمایش موجودی، قیمت، و SKU
- ✅ رنگ‌بندی بر اساس موجودی (سبز/زرد/قرمز)

#### کلیدهای میانبر:
- `⬆️ Arrow Up`: حرکت به بالا در لیست
- `⬇️ Arrow Down`: حرکت به پایین در لیست
- `Enter`: افزودن کالای انتخاب شده
- `Esc`: بستن dropdown و پاک کردن جستجو

---

### 2. جریان فوکوس (Keyboard Flow)

#### مسیر فوکوس:
```
جستجوی کالا → Enter → تعداد → Enter → قیمت فی → Enter → جستجوی کالا
```

#### رفتار هوشمند:
- ✅ **Select on Focus**: با فوکوس شدن، تمام متن انتخاب می‌شود
- ✅ **Backspace Logic**: در فیلد خالی، Backspace به فیلد قبلی برمی‌گردد
- ✅ **Auto Focus**: بعد از افزودن کالا، فوکوس روی تعداد می‌رود

---

### 3. ویرایش قیمت کل (Ctrl+B)

#### نحوه استفاده:
1. `Ctrl + B` را بزنید
2. قیمت کل مورد نظر را وارد کنید
3. سیستم خودکار تخفیف کلی را محاسبه و اعمال می‌کند

#### مثال:
```
جمع اقلام: 1,000,000 ریال
قیمت کل دلخواه: 900,000 ریال
→ تخفیف کلی: 100,000 ریال (خودکار)
```

---

### 4. انیمیشن‌های Smooth

#### Dropdown:
- انیمیشن slideDown با cubic-bezier
- مدت زمان: 150ms
- حس کشسان و حرفه‌ای

#### Focus States:
- Ring effect با رنگ آبی
- Transition smooth روی تمام inputها
- Scale effect روی آیتم‌های highlighted

## 🎯 تجربه کاربری

### سرعت ثبت فاکتور:
با این قابلیت‌ها، اپراتور می‌تواند:
- ✅ بدون نگاه به مانیتور کار کند
- ✅ فقط با کیبورد فاکتور صادر کند
- ✅ در کمتر از 30 ثانیه یک فاکتور 5 قلمی ثبت کند

### مثال عملی:
```
1. تایپ "لپ" → Enter (افزودن لپ‌تاپ)
2. تایپ "2" → Enter (تعداد 2)
3. Enter (قیمت پیش‌فرض تایید)
4. تایپ "موب" → Enter (افزودن موبایل)
5. تایپ "1" → Enter
6. Enter
7. Ctrl+B → "5000000" → Enter (تخفیف کلی)
8. ثبت نهایی
```

## 🔧 جزئیات فنی

### Refs استفاده شده:
```typescript
searchInputRef: HTMLInputElement
resultsRef: HTMLDivElement
quantityRefs: { [itemId: string]: HTMLInputElement }
priceRefs: { [itemId: string]: HTMLInputElement }
discountRefs: { [itemId: string]: HTMLInputElement }
```

### Event Handlers:
- `handleSearchKeyDown`: مدیریت کیبورد در جستجو
- `handleQuantityKeyDown`: مدیریت Enter و Backspace در تعداد
- `handlePriceKeyDown`: مدیریت Enter و Backspace در قیمت
- `handleDiscountKeyDown`: مدیریت Enter در تخفیف

### Auto-scroll Logic:
```typescript
activeElement.scrollIntoView({ 
  block: "nearest", 
  behavior: "smooth" 
});
```
