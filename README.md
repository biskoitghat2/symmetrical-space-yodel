## 📖 معرفی

سیستم حسابفلو یک نرم‌افزار حسابداری جامع و کامل برای مدیریت:
- 💰 حسابداری و مالی
- 📦 انبارداری
- 👥 مشتریان و کاردکس
- 🏭 تولید و کارگاه
- 🔧 رسید تعمیرات
- 💳 چک‌ها و بانک‌ها
- 📊 گزارش‌گیری

## 🚀 نصب و راه‌اندازی

### پیش‌نیازها:
- Node.js (v18 یا بالاتر)
- npm یا yarn
- Rust (برای بیلد Tauri)

### نصب:
```bash
# نصب وابستگی‌ها
npm install

# اجرای برنامه در حالت توسعه
npm run dev

# بیلد برای Production
npm run tauri build
```

## 🛠️ تکنولوژی‌ها

- **Frontend:** React + TypeScript + Vite
- **UI:** Tailwind CSS + Lucide Icons
- **State Management:** Zustand
- **Database:** SQLite (via Tauri)
- **Desktop:** Tauri v2
- **Date:** Jalali (Persian Calendar)
