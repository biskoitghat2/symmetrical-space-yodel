## 📋 مراحل ساخت فایل نصبی

### مرحله 1: آماده‌سازی آیکون (اختیاری)

اگر می‌خواهید آیکون سفارشی داشته باشید:

1. یک تصویر PNG با ابعاد 512x512 پیکسل آماده کنید
2. نام آن را `app-icon.png` بگذارید
3. در ریشه پروژه قرار دهید
4. این دستور را اجرا کنید:

```bash
npm run tauri icon app-icon.png
```

این دستور آیکون‌های مورد نیاز Windows را خودکار می‌سازد.

### مرحله 2: تست نهایی

قبل از build، مطمئن شوید همه چیز کار می‌کند:

```bash
npm run tauri dev
```

تست کنید:
- ✅ افزودن کالا
- ✅ ثبت فاکتور
- ✅ کاردکس مشتری
- ✅ پشتیبان‌گیری
- ✅ بستن سریع برنامه

### مرحله 3: ساخت فایل نصبی

```bash
npm run tauri build
```

این دستور:
1. Frontend را build می‌کند
2. Backend Rust را compile می‌کند
3. فایل نصبی Windows را می‌سازد

**زمان تقریبی**: 5-10 دقیقه (اولین بار)

## 📦 فایل‌های خروجی

بعد از build موفق، فایل‌ها در این مسیر ایجاد می‌شوند:

```
src-tauri/target/release/bundle/
```

### فایل‌های Windows:

#### 1. MSI Installer (توصیه می‌شود)
```
msi/HesabFlow_1.0.0_x64_en-US.msi
```
- نصب استاندارد Windows
- قابل حذف از Control Panel
- حجم: 5-10 MB

#### 2. NSIS Installer
```
nsis/HesabFlow_1.0.0_x64-setup.exe
```
- نصب سریع‌تر
- حجم کمتر
- حجم: 5-10 MB

#### 3. فایل اجرایی (Portable)
```
../release/HesabFlow.exe
```
- بدون نیاز به نصب
- قابل اجرا از USB
- حجم: 3-5 MB

## 🧪 تست فایل نصبی

### تست 1: نصب

1. فایل MSI را روی کامپیوتر دیگری (یا VM) کپی کنید
2. دابل کلیک کنید
3. مراحل نصب را طی کنید
4. برنامه را از Start Menu اجرا کنید

### تست 2: عملکرد

1. کالا اضافه کنید
2. برنامه را ببندید
3. دوباره باز کنید
4. کالا باید هنوز موجود باشد ✅

### تست 3: پشتیبان‌گیری

1. به تنظیمات بروید
2. مسیر دیتابیس را ببینید (باید چیزی شبیه این باشد):
   ```
   C:\Users\[نام کاربری]\AppData\Roaming\com.hesabflow.app\hesabflow.db
   ```
3. دکمه "ایجاد پشتیبان" را بزنید
4. مسیر ذخیره را انتخاب کنید
5. فایل `.db` ذخیره می‌شود ✅

### تست 4: بازگردانی

1. داده‌های جدید اضافه کنید
2. دکمه "بازگردانی" را بزنید
3. فایل backup قبلی را انتخاب کنید
4. برنامه restart می‌شود
5. داده‌های قدیمی برمی‌گردند ✅

## 🐛 رفع مشکلات احتمالی

### مشکل 1: خطای Build

**علامت**:
```
Error: Failed to build
```

**راه حل**:
```bash
cd src-tauri
cargo clean
cargo build --release
cd ..
npm run tauri build
```

### مشکل 2: آیکون نمایش داده نمیشه

**راه حل**:
```bash
npm run tauri icon app-icon.png
```

مطمئن شوید فایل `app-icon.png` در ریشه پروژه است.

### مشکل 3: Windows Defender فایل را بلاک می‌کند

**راه حل**:
1. Windows Security را باز کنید
2. Virus & threat protection → Manage settings
3. Exclusions → Add or remove exclusions
4. پوشه پروژه را اضافه کنید

### مشکل 4: دیتابیس ایجاد نمیشه

**بررسی**:
1. `src-tauri/tauri.conf.json` را باز کنید
2. مطمئن شوید `"withGlobalTauri": true` است
3. دوباره build بگیرید
