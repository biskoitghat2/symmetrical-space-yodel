## 🧪 سناریوهای تست:

### تست ۱: فاکتور فروش نقدی ساده
```
پیش‌شرط:
- مشتری A: مانده = 0
- کالا X: موجودی 100، قیمت فروش 1000
- حساب بانکی: مانده 50000

مراحل:
1. ثبت فاکتور فروش
2. مشتری: A
3. کالا X: 10 عدد × 1000 = 10000
4. پرداخت: نقدی 10000 (حساب بانکی)

نتیجه مورد انتظار:
✅ موجودی کالا X: 90 (100 - 10)
✅ مانده مشتری A: 0 (فروش +10000، پرداخت -10000)
✅ مانده حساب بانکی: 60000 (50000 + 10000)
✅ تراکنش بانکی ثبت شده
✅ 2 تراکنش در کاردکس مشتری (فاکتور + پرداخت)
```

### تست ۲: فاکتور فروش نسیه
```
پیش‌شرط:
- مشتری B: مانده = 0
- کالا Y: موجودی 50

مراحل:
1. ثبت فاکتور فروش
2. مشتری: B
3. کالا Y: 5 عدد × 2000 = 10000
4. پرداخت: نسیه (بدون پرداخت)

نتیجه مورد انتظار:
✅ موجودی کالا Y: 45 (50 - 5)
✅ مانده مشتری B: +10000 (بدهکار)
✅ 1 تراکنش در کاردکس (فاکتور)
✅ هیچ تراکنش بانکی ثبت نشده
```

### تست ۳: فاکتور فروش با چک

```
4. پرداخت: چک 15000

✅ 2 تراکنش در کاردکس (فاکتور + چک)
```
