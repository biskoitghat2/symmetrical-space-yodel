  - ثبت تراکنش فاکتور

  - اگر پرداخت نقدی:
    - ثبت تراکنش پرداخت
    - آپدیت balance مشتری
    - آپدیت موجودی بانک
    - ثبت تراکنش بانکی
  - اگر چک:
    - ثبت تراکنش چک
    - آپدیت balance مشتری

## 🎯 تست‌های پیشنهادی

برای اطمینان از کارکرد صحیح، این تست‌ها را انجام دهید:

1. **کاردکس مشتری**: افزودن تراکنش → بررسی balance
2. **فاکتور فروش**: ثبت فاکتور → بررسی موجودی کالا و balance مشتری
3. **پرداخت نقدی**: فاکتور با پرداخت → بررسی موجودی بانک
4. **چک**: ثبت و پاس چک → بررسی balance و موجودی بانک
5. **تولید**: تکمیل تولید → بررسی موجودی محصول
6. **رسید تعمیرات**: افزودن قطعه → بررسی موجودی انبار
7. **تبدیل به فاکتور**: تبدیل رسید → بررسی فاکتور و وضعیت رسید

همه این تست‌ها باید موفق باشند و داده‌ها بعد از restart حفظ شوند.
