```
  ↓
[10ms] 🎨 صفحه لودینگ HTML نمایش داده می‌شود
  |     (لوگو + اسپینر + "در حال بارگذاری...")
  |
[10-500ms] مرورگر فایل JS (2MB) را دانلود و parse می‌کند
  |
[500ms] ⚛️ React بالا می‌آید
  ↓
[500ms] 🔄 صفحه لودینگ HTML با fade out حذف می‌شود
  ↓
[500ms] 🎨 SplashScreen React نمایش داده می‌شود (1 ثانیه)
  ↓
[1500ms] 🎨 LoadingScreen React نمایش داده می‌شود
  |     (نوار پیشرفت + مراحل)
  |

  |     - راه‌اندازی دیتابیس (5%)

  |

```

### 4. `store/dataStore.ts` ⭐ تغییر
**تغییرات:**
- ✅ اضافه شدن callback به `loadAllData`
- ✅ گزارش پیشرفت بعد از هر مرحله

**قبل:**
```typescript
loadAllData: () => Promise<void>
```

**بعد:**
```typescript
loadAllData: (onProgress?: (step: string, progress: number) => void) => Promise<void>
```

### 5. `App.tsx` ⭐ تغییر
**تغییرات:**
- ✅ اضافه شدن state برای tracking پیشرفت
- ✅ استفاده از LoadingScreen به جای div ساده
- ✅ ارسال callback به loadAllData

**کد اضافه شده:**
```typescript
const [loadingStep, setLoadingStep] = useState('database');
const [loadingProgress, setLoadingProgress] = useState(0);

await loadAllData((step, progress) => {
  setLoadingStep(step);
  setLoadingProgress(progress);
});

return <LoadingScreen currentStep={loadingStep} progress={loadingProgress} />;
```
