# راهنمای تست دیتابیس

## وضعیت فعلی
✅ دیتابیس به درستی پیاده‌سازی شده است
✅ تمام متدهای CRUD به صورت async هستند
✅ سیستم نوتیفیکیشن فعال است

## مشکل گزارش شده
- هیچ چیز سیو نمیشه
- نوتیفیکیشن نمیاد

## مراحل تست

### مرحله 1: بررسی Console
لطفاً برنامه را اجرا کنید و Console را باز کنید (F12 در مرورگر یا DevTools در Tauri).

باید این پیام‌ها را ببینید:
```
🔄 Step 1: Starting initialization...
🔄 Step 2: Initializing database...
🔄 Loading database...
✅ Database loaded
🔄 Creating tables...
✅ Tables created
✅ Database initialized successfully
✅ Step 3: Database initialized
🔄 Step 4: Checking migration...
📊 Step 5: Migration needed: true/false
📥 Step 8: Loading data from database...
✅ Step 9: Data loaded successfully
✅ Step 10: Setting initialized to true
🎉 Initialization complete!
```

### مرحله 2: تست افزودن کالا
1. به بخش "مدیریت انبار" بروید
2. یک کالای جدید اضافه کنید
3. در Console باید ببینید:
```
✅ All data loaded from database
```

### مرحله 3: بررسی خطاها
اگر خطایی وجود دارد، در Console نمایش داده می‌شود:
```
❌ Failed to initialize app: [error message]
❌ Database initialization failed: [error message]
❌ Failed to load data: [error message]
```

## احتمالات مشکل

### 1. دیتابیس اصلاً initialize نمیشه
**علت**: برنامه در مرورگر اجرا میشه نه Tauri
**راه حل**: مطمئن شوید با دستور `npm run tauri dev` اجرا کردید

### 2. دیتابیس initialize میشه ولی save نمیکنه
**علت**: خطا در متدهای DatabaseService
**راه حل**: Console را چک کنید برای خطاهای مربوط به SQL

### 3. نوتیفیکیشن نمیاد
**علت**: هیچ شرطی برای نوتیفیکیشن برقرار نیست
**مثال**: 
- موجودی کالا باید صفر یا کمتر از حد هشدار باشد
- چک باید سررسید امروز یا گذشته باشد
- بدهی مشتری باید بیش از 500 میلیون باشد

## تست دقیق

### تست 1: افزودن کالا
```
1. به انبار بروید
2. کالای جدید اضافه کنید با نام "تست"
3. برنامه را ببندید (دکمه X)
4. دوباره باز کنید
5. آیا کالای "تست" هنوز وجود دارد؟
```

### تست 2: نوتیفیکیشن موجودی
```
1. کالایی با موجودی 0 اضافه کنید
2. آیکون زنگ (🔔) در نوار بالا را چک کنید
3. باید نوتیفیکیشن "اتمام موجودی" نمایش داده شود
```

### تست 3: بستن سریع برنامه
```
1. کالایی اضافه کنید
2. بلافاصله دکمه X را بزنید
3. برنامه باید فوراً بسته شود (بدون فریز)
4. دوباره باز کنید و کالا را چک کنید
```

## اگر مشکل همچنان وجود دارد

لطفاً این اطلاعات را به من بدهید:
1. تمام پیام‌های Console (کپی کامل)
2. آیا با `npm run tauri dev` اجرا کردید؟
3. آیا برنامه در مرورگر باز میشه یا پنجره Tauri؟
4. آیا خطایی در Console هست؟
5. دقیقاً چه عملیاتی انجام دادید که save نشد؟
